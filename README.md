# pip-helper
可以帮助不会使用pip安装库的人准备

# 使用方式
首先在**https://github.com/dengrb1/pip-helper**里面下载最新的版本

然后再打开刚刚下载好的文件，然后想平时安装程序一样使用本程序吧！！！
